# Laboratory Activity 3.1 - Introduction to JSX and Mobile Layouts – Sign Up View

A React (Vite) project implementing a mobile Sign Up View using JSX and CSS Flexbox.

## Objectives
- Construct UI with JSX
- Implement Flexbox for Mobile Layouts
- Apply Form Input Handling (Visual)

## Getting Started

```bash
npm install
npm run dev
```

Open your browser Developer Tools and use a mobile emulator (e.g., iPhone 13 or Pixel 5) to view the application.
